== Description ==

This plugin alows the WordPress admin to customise the wp-login.php logo.

== Installation ==

1. Extract "custom-wordpress-login-image.zip"
2. Upload the "custom-wordpress-login-image" Folder to the Plugin Directory
3. Activate the plugin through the WordPress admin "Plugins" menu
4. Navigate to the WordPress customizer site identity section.
5. Upload custom image.

== Frequently Asked Questions ==

= I have activated the plugin, where do I find the settings? =

You can find the settings for this plugin in the WordPress customizer, under the "site identity" section.
